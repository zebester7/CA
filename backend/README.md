# Sterling & Vance Backend API

Node.js + Express backend for the Sterling & Vance financial advisory platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Routes

- `/api/health` - Health check
- `/api/auth` - Authentication endpoints
- `/api/consultations` - Consultation booking endpoints
- `/api/services` - Services endpoints
- `/api/team` - Team endpoints
- `/api/case-studies` - Case studies endpoints
